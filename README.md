# 지원 기능
* 웹툰 부분, 전체 다운로드
* 성인 웹툰 다운로드
* 웹툰 받을시 고유번호 or 검색으로 다운 가능

# ⛔Warning
웹툰을 받고 반드시 소장용으로만 사용해주세요.  
웹툰의 경우 엄연한 저작물이기에 무단배포시 법적 처벌을 받을 수 있습니다.
